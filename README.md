# Artificial Intelligence – Computer Vision w Pythonie

Repozytorium zawiera materiały, notatki oraz projekty stworzone podczas kursu Artificial Intelligence – Computer Vision w języku Python, prowadzonego przez Pawła Krakowiaka na platformie Udemy.
Celem kursu jest wprowadzenie do zagadnień związanych z widzeniem komputerowym (Computer Vision) oraz praktyczne wykorzystanie bibliotek Python do analizy i przetwarzania obrazów.

## Cele kursu
- Zrozumienie podstaw Computer Vision
- Praca z obrazami i wideo w Pythonie
- Wykrywanie obiektów i konturów
- Przetwarzanie obrazu (filtry, transformacje)
- Wprowadzenie do sztucznej inteligencji w analizie obrazu

## Technologie i narzędzia
- Python
- OpenCV
- NumPy
- Matplotlib

## Zakres materiału
- Wczytywanie i wyświetlanie obrazów
- Operacje na pikselach
- Filtry i transformacje (blur, thresholding)
- Detekcja krawędzi (Canny)
- Kontury i ich analiza
- Praca z kamerą i strumieniem wideo
- Podstawy rozpoznawania obiektów
